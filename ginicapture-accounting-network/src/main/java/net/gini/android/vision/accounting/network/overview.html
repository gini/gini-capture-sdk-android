Implementation using the Gini Accounting API of the network related tasks required by the Gini Capture SDK.

Adding this library along with the Gini Capture SDK to your application is the quickest way to integrate document scanning with Gini's Accounting API.

In order for the Gini Capture SDK to use this networking library pass the instances to the GiniVision.Builder when creating a new GiniVision.

Important:You must not enable multi-page in the Gini Capture SDK when using it with this networking library. The multi-page feature is currently only supported by the default networking library.

Requirements

Android 4.4+ (API Level 19+)
