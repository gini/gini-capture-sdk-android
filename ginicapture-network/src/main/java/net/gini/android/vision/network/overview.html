The Gini Vision Network Library for Android provides default implementation of the network related tasks required by the Gini Vision Library.

Adding this library along with the Gini Vision Library to your application is the quickest way to integrate invoice scanning.

In order for the Gini Vision Library to use the default implementations pass the instances to the GiniVision.Builder when creating a new GiniVision.

Requirements

Android 4.4+ (API Level 19+)
